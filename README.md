# EaseLease

EaseLease is a lease finder web application developed to help new and continuing Masterâ€™s students easily search, list, and manage rental properties.

The platform provides a centralized solution for property discovery, virtual tour scheduling, and lease agreement handling, reducing the complexity of the traditional rental process.

---

## Key Features

- User registration and authentication
- Profile creation and management
- Property listing for rent or sale
- Property search and filtering
- Virtual tour scheduling
- Lease agreement creation and PDF download
- Secure data storage and access control

---

## Technology Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend & Cloud Services
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- Google Cloud Platform

---

## System Overview

- The application is built as a single-page application (SPA) using React.
- Firebase is used for authentication, database management, and file storage.
- Google Cloud infrastructure ensures scalability and reliability.
